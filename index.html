Hasznos fogalmak

Bootstrap
A Bootstrap egy CSS (Cascading Style Sheet) keretrendszer, előre megírt CSS fájlok alkotják. Ahhoz, hogy használni tudjuk a stílusokat, be kell ágyaznunk a CSS fájlt az oldalunkba.
CSS
(CSS - Cascading Style Sheet, lépcsőzetes, vagy rangsorolt stílusalapok) HTML-lel helyezünk el szöveget az oldalakon, és a CSS stílusleíró nyelv segítségével adunk hozzá stíluselemeket, mint például színek, betűtípus, megjelenés, elrendezés.
Hálózati kártya
A hálózati kártyák (NIC – Network Interface Card) nyomtatott áramköri lapok, amelyek a számítógépek alaplapján vagy perifériáján levő busz csatlakozójába illeszthetők. Feladatuk a számítógép és a hálózat átviteli közegének összekapcsolása. A világ minden hálózati kártyája egyedi azonosítóval, az ún. közegelérési (Media Access Control, MAC) címmel (MAC address) rendelkezik.
Háttértár HDD
(HDD - Hard disk drive) Belső merevlemez, melynek feladata az adatok tárolása. Mechanikusan, a mágnesesség elvén működő háttértár. Relatíve olcsó, nagy kapacitású, közepes sebességű (50-150 MB/s) háttértárolási lehetőséget biztosít. A háttértár a betáp lekapcsolásakor is megőrzi az adatokat. Ezt a merevlemez típust az egyre nagyobb kapacitású, kisebb fogyasztású SSD - Solid state drive háttértár lassan kiszorítja a piacról.
Háttértár SSD
(SSD - Solid state drive) Félvezetős memóriát használó adattároló eszköz. Nem tartalmaz mozgó alkatrészeket, hangtalan, elenyésző a hőtermelése. Kevesebb energiát fogyaszt, mint a HDD eszközök, ennek ellenére nagyobb írási, olvasási sebességre képes. Drágább, mint a hagyományos merevlemezek.
HTML5
(HTML- HiperText Markup Language) Hiperszöveges jelölőnyelv, internetes szabvány, melyet weboldalak készítéséhez fejlesztettek ki. Sokféleképpen neki lehet állni egy weboldal készítésének, de egy HTML vázra alapvető szükség lesz. A weben a böngésző az a program, amely egy HTML-lel létrehozott dokumentumot képes megnyitni és megjeleníteni. A HTML legfrissebb verziója a HTML5.
HTTP
(HTTP - Hypertext Transfer Protocol, hiperszöveg átviteli protokoll): adatátviteli protokoll. Kliens-szerver modellen alapuló kérés-válasz protokoll.
HTTPS
A HTTP secure, a HTTP biztonságos verziója. A HTTPS-t a Netscape böngészőgyártó fejlesztette ki, nagyobb biztonsági szükségletű weboldalakhoz, például banki oldalakhoz. A weboldalak SSL (Secured Sockets Layer) protokollon keresztül titkosított formában kerülnek átvitelre. Amennyiben egy weboldal SSL tanúsítvánnyal rendelkezik akkor a linkje https-sel kezdődik sima http helyett, és egy zárt lakat látszódik a böngésző címsorában.
Internet
1990-ben csatlakozott fel az első számítógép a globális számítógépes hálózatra, az internetre, ahol a számítógépek az internetprotokoll (IP) segítségével kommunikálnak egymással. A hálózatok az információcseréhez egységes szabványokat használva működnek együtt. 1991 Tim Berners-Lee-nek és csapatának köszönhetően kifejlesztésre került a világháló (World Wide Web, www), az interneten működő, hiperhivatkozásokkal összekötött dokumentumok rendszere. 1993-ban a Európai Nukleáris Kutatási Szervezet, azaz a European Organization for Nuclear Research (CERN) bejelentette, hogy a világháló mindenki számára szabad és ingyenes.
IP-cím
Egyedi hálózati azonosító, amelyet az IP segítségével kommunikáló számítógépek egymás azonosítására használnak. Minden internetre kapcsolódó számítógépnek van IP-címe, de egy-egy konkrét cím nem kötődik feltétlenül egy-egy géphez. Az IP címet az operációs rendszer által felismert és használt hálózati kártyához rendeljük hozzá. Az IP-cím számos országban, többek között Magyarországon is személyes adatnak számít.
Port
TCP kommunikáció esetében az alkalmazások azonosításához, megkülönböztetéséhez is szükséges egy azonosító. Ezt nevezzük portszámnak vagy socket number-nek. Sok mindent azonosítani kell ahhoz, hogy az alkalmazások egymással kommunikálni tudjanak, például melyik interface, melyik IP cím, melyik protokoll és melyik alkalmazás, ami azt előállította és kinek kell megkapnia. Ezért egy port azonosítóra is szükségünk van. Erre használunk egy maximum 16 bit-es azonosítót, ahol decimálisan 65.536 különböző azonosító lehetséges. A 255 alatti számok nyilvános, közismert alkalmazásokhoz tartoznak. A 255-től 1023-ig terjedő számokat a piaci alkalmazásokat fejlesztő cégek használhatják. (Az első játék, ami saját portszámot kapott a DOOM volt: 666) Az 1023 feletti tartományra nem vonatkozik szabvány. Bárki, bármilyen azonosítót kiválaszthat 1023 felett.
Router (forgalomirányító)
Feladata a bejövő csomagok megvizsgálása, a legjobb hálózati útvonal kiválasztása, és a csomagok átkapcsolása a megfelelő kimenő portra. A router beül különböző TCP/IP alhálózatok közé, és átjáróként funkcionál közöttük. Lehet tetszőleges számú interface, akár LAN akár WAN hálózat is. Ismerni fogja ezeket a hálózatokat, minden egyes interface-en lesz egy az adott, kezelt hálózatba tartozó egyedi IP cím, ez lesz a default gateway címe (alapértelmezett átjáró cím) a klienseknek.
Switch
A mai modern hálózatainkban az egyik alapvető aktív eszköz a kapcsoló vagy SWITCH. Célja, hogy minden végpontnak egyenrangú hozzáférést biztosítsunk a hálózathoz. A rácsatlakoztatott eszközök között adatáramlást valósít meg. Az adatkapcsolati rétegben működő switch. a MAC address-ek alapján végzi a feladatát.
TCP/IP
Az internet hivatkozási modellje. Elsősorban a logikai hálózatokhoz tartozó kommunikációs megoldás. Több részprotokoll együttműködéséről beszélünk. Az egyik ledarabolja a kommunikációt, a másik megcímzi ezeket. A TCP/IP protokollgyűjtemény az elmúlt negyven év során több módosításon is átesett. Napjainkban a 4-es és a 6-os változatot használjuk. A kommunikáció jelentős része még a TCP/IP V4-es verzión keresztül történik. A TCP/IP a világ bármely két pontján (vagy azon kívül) lévő számítógépek között biztosítja az adatkommunikációt. Az Amerikai Védelmi Minisztérium definiálta, mert egy olyan hálózatot kívánt létrehozni, amely minden körülmények között – még egy atomháború esetén is – működőképes marad. Négy réteget tartalmaz: az alkalmazási réteget, a szállítási réteget, az internet réteget és a hálózati réteget.
Visual Studio Code (VSC)
A Microsoft által fejlesztett Integrated Development Environment (IDE), Integrált Fejlesztői Környezet. A segítségével tudunk programkódokat szerkeszteni. Kiváló támogatást nyújt a webes fejlesztésekhez, ingyenesen letölthető és több ezer kiegészítő elérhető hozzá. Ezekkel a kiegészítőkkel megtaníthatjuk a VSC-t új nyelvekre és technológiákra.
VSC - Live Server Extension
Ezzel a kiegészítővel a mentés után azonnal láthatóvá válik a böngészőben a programkódok által létrehozott eredményt.
